# MYA Parts Project Plan

## Goal
Build a B2B E-commerce/Catalog for Forklift Parts with a quote cart, tolerant search, and 3D/multiple image support, using Next.js and MySQL.

## Tasks
- [x] Task 1: Scaffold Next.js project with Tailwind CSS. → Verify: Project runs locally.
- [x] Task 2: Configure MySQL database and Prisma schema. → Verify: Schema is migrated and Prisma Client generated.
- [x] Task 3: Build the industrial design system (Theme, Navbar, Footer, Buttons). → Verify: Basic layout renders correctly.
- [x] Task 4: Implement the Backend API for Catalog (Tolerant Search algorithm). → Verify: API endpoints return correct products even with formatting differences.
- [x] Task 5: Build the Admin Panel (Auth + CRUD Products/Tags). → Verify: Admin can create a product with multiple images and 3D viewers.
- [x] Task 6: Build the Frontend public catalog (Home, Categories, Product Page). → Verify: Pages render correctly and SEO tags are present.
- [x] Task 7: Implement the Quote Cart functionality (State management + API connection). → Verify: User can add items, view cart, and submit quote.
- [x] Task 8: Run checklist scripts and prepare for deployment. → Verify: 0 errors in linting and build.

## Done When
- [x] Next.js project is fully functional locally.
- [x] Admin panel allows complete management of the catalog, including multi-image/3D.
- [x] Public site allows robust searching and adding to a quote cart.
- [x] Code is modular, clean, and ready for GitHub/Hostinger CI/CD.
